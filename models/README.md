# Model weights

Place six fine-tuned Ultralytics checkpoints here:

- `powerline_yolo11n.pt`, `powerline_yolo11s.pt`;
- `fracture_yolov8_fast.pt`, `fracture_yolov8l.pt`;
- `vehicle_yolo11n.pt`, `vehicle_yolo11s.pt`.

The training notebook exports checkpoints with these exact names. Weight files
are excluded from Git; use Git LFS, a release asset, or private object storage.
